# Social MERN v2 - Vite Frontend + Updated Backend

What's new:
- Frontend rebuilt using Vite + React with a flashy, over-the-top UI (no external UI libs required).
- Frontend features: search, create, edit, delete, like, comment, follow/unfollow, people search, user profile and posts.
- Backend fixes: correct multer dependency (1.4.5-lts.1) and new endpoints for edit/delete/search posts.
- Zip includes ready-to-run backend and frontend.

Run:
1. Start backend:
   cd backend
   npm install
   cp .env.example .env
   npm run dev

2. Start frontend:
   cd frontend
   npm install
   npm run dev

Default API URL: http://localhost:5000
